You are welcome to Flight Accademy!
In order to pass the test you need to find the RED landing base and land on it.
ENGINE START
To take off you must start engine. To do it press the GREEN button on the left (Engine Start). Once the button has been pressed, wait for the starting procedure to be completed (green light on).
When the green light is on, you are ready to take off.
